BRĪDINĀJUMS: Tīmekļa strādņu testa lietotnes nestrādā, ja tās tiek palaistas no html faila. Lai tīmekļa strādņi darbotos, lietotnes nepieciešams palaist lokālā serverī.

Šajā testa lietotnē tiek demonstrēts salīdzinājums starp lielapjoma datu apstrādi galvenajā pavedienā un šo datu sūtīšanu tīmekļa strādnim apstrādei. Testa lietotnē ir iespējams izvēlēties - vai datus sūtīt kā JSON objektu vai kā simbolu virkni, kā arī iespējams izvēlēties, vai aprēķini tiek veikti tīmekļa strādnī vai galvenajā pavedienā.

Ātrdarbības mērījumu un aprēķināto skaitli iespējams redzēt izstrādātāja rīkos Console sadaļā.
